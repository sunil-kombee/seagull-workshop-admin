# Seagull Admin

This is a Next.js + React + TypeScript project.

## Getting Started

### Prerequisites
- [pnpm](https://pnpm.io/) (v8 or higher)

### Install dependencies
```bash
pnpm install
```

### Run the development server
```bash
pnpm dev
```

### Build for production
```bash
pnpm build
```

### Start the production server
```bash
pnpm start
```
